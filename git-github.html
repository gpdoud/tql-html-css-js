Overview

Git & GitHub are source control program that work on the desktop and in the cloud.

Initialize a repository

git init

Create a repository in the current directory.

Stage a file or files

git add { . | filename | filespec }

Designates a snapshot of file or collection of files for storing into a repository.

Commit all staged file into the repository

git commit -m "message"

Stores all the staged files into the repository with a message
